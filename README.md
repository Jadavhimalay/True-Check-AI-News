## 📌 Description
True Check AI News is a web-based application that helps users verify whether a news article is real or fake using basic content analysis.

---

## 🌐 Live Demo
👉 https://jadavhimalay.github.io/True-Check-AI-News/

---

## 🚀 Features
- 📝 Enter or paste news content  
- 🔍 Analyze text for authenticity  
- ✅ Displays result (Real / Fake)  
- ⚡ Fast and simple interface
